# Contributing to GITSEARCH

## Types of Contributions

- **Indexation** — Add the GITSEARCH badge to your README to enrich the ecosystem
- **Semantic Dictionary** — Improve synonyms, add new terms
- **Domains** — Propose new tech domains with their keywords
- **Bug Reports** — Open a GitHub Issue with reproduction steps

## Standards

- No npm in runtime (Tailwind via CDN only)
- No backend, no server — everything must work statically
- WCAG 2.1 AA for accessibility
- SVG < 50KB for badges, < 150KB for animations
- `prefers-reduced-motion` on all CSS and SMIL animations
